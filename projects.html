Teaching Accessibility in CS


Machine Learning Fairness Education


This project sought to teach students in a Natural Language Processing course about the biases those models can inherit. We did this through a mixture of lecture, programming activities, and a project discussing Machine Learning fairness in different situations. Additionally, we interviewed students and analyzed their responses to determine the results of the interventions.

This project was in collaboration with Dr. Yudong Liu. Parts of this work are supported by the SIGCSE Special Projects Grant award 56004A.
