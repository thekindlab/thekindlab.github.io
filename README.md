# The KIND Lab Website

The official website for **The KIND Lab** (Komputing for INclusion & Disability) at Western Washington University's Computer Science Department.

## About

The KIND Lab focuses on accessibility research and inclusive design in computing. This website showcases the lab's research, publications, team members, and news.

## Pages

- **Home** - Lab introduction, culture, and vision
- **People** - Faculty and student profiles
- **Publications** - Research publications
- **Current Projects** - Ongoing research projects
- **News** - Lab announcements
- **Contact** - Contact information

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- Font Awesome 6.4.0
- Academicons 1.9.3

## Accessibility

To ensure accessibility with screen readers I added the following: 

- Skip-to-main-content navigation (holding option and then tab on a Mac)
- ARIA attributes for interactive elements
- Keyboard navigation support
- WCAG AA compliant color contrast
- Semantic HTML structure (h tags rather than div)
